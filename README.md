# Calculadora em Python 🧮

Olá! Este é um projeto prático desenvolvido em Python com o objetivo de consolidar conceitos fundamentais de lógica de programação, organização de código e tratamento de dados.

 🛠️ Tecnologias e Conceitos Aplicados:
* **Linguagem:** Python
* **Funções (`def`):** Para modularizar e organizar cada operação matemática.
* **Estruturas Condicionais (`if`, `elif`, `else`):** Para direcionar a escolha da operação pelo usuário.
* **Estruturas de Repetição (`while`):** Para permitir que o usuário faça múltiplos cálculos sem precisar reiniciar o programa.

## 📈 Funcionalidades do Sistema:
* Realiza as 4 operações básicas: Soma, Subtração, Multiplicação e Divisão.
* Tratamento de erros (como validação para impedir a divisão por zero).
* Menu interativo e limpo para o usuário.
